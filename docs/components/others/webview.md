# 网页容器
内嵌外部站点，可用 iframe 来实现。


## 基本用法

::: leivii
``` js
{
    "$id": "geek-web-view-H6oSn2HDpoab",
    "$type": "geek-web-view",
    "$visible": true,
    "src": "http://172.18.1.80/static/eos/#/login",
    "$classes": null,
    "name": "iframe",
    "scrolling": "yes"
}
```
:::


## Props

<div class="props">

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| ---- | ---- | ---- | ---- | ---- |
| src | 网页地址 | String | - | - |
| name | 原生name属性 | String | - | - |
| width | 宽度 | String | - | 100% |
| height | 高度 | String | - | 100% |
| scrolling | 滚动条 | String | auto / yes / no | - |

</div>
