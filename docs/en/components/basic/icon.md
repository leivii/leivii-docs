# 图标

常用的图标。

## 基础用法

基础的图标用法。

::: leivii
``` js
{
  '$id': 'geek-container-oAyRkGYugqxu',
  '$type': 'geek-container',
  '$visible': true,
  '$body': [
      {
          '$id': 'geek-icon-YdgLgrXbHMFt',
          '$type': 'geek-icon',
          '$visible': true
      },
      {
          '$id': 'geek-icon-OfK9lJPozeI2',
          '$type': 'geek-icon',
          '$visible': true,
          'iconClass': 'download',
          '$classes': null,
          'title': '提示',
          'size': 0
      },
  ],
  'grids': [],
  '$classes': 'm-md',
  'limit': 0,
  'gutter': 20,
  'rows': 1,
  'cols': 5,
  'mode': 'vertical'
}
```
:::

## Props

<div class='props'>

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| ---- | ---- | ---- | ---- | ---- |
| title | 提示文案 | string | - | - |
| iconClass | 图标类名 | string | - | - |
| size | 尺寸 | Number | - | - |

</div>
